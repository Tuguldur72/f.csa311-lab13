# Personal Task Tracker - Project Brief

## Goal
Хэрэглэгчийн өдөр тутмын ажлыг удирдах Node.js-д суурилсан REST API хөгжүүлэх.

## Target Features (Үндсэн функцууд)
1. **Task CRUD**: Ажил үүсгэх, унших, засах, устгах.
2. **Persistence**: Бүх өгөгдлийг MongoDB баазад хадгалах.
3. **Filtering**: Ажлуудыг дуусах хугацаа (due date) болон ач холбогдлоор (priority) шүүх.
4. **Validation**: Оруулсан өгөгдлийг шалгах (жишээ нь: хоосон гарчиг зөвшөөрөхгүй).

## Success Criteria (Амжилтын шалгуур)
- 3-аас доошгүй feature бүрэн ажиллах.
- 10-аас доошгүй unit test амжилттай давах.
